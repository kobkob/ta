Funcionalidades do ta

ta é um website na iternet que permite buscas em fontes abstratas de dados.

A implementacao inicial usa um mashup com o google maps mas deve sempre haver uma forma clara e simples de aplicar ganchos para outras fontes de dados a critério do usuário.

  • Abertura (Welcome)
A interface de abertura usa um mashup do panoramio para apresentar o fundo de tela com imagens aleatórias do Brasil. A idéia é criar um set de imagens e carregar na abertura. Durante as buscas a tag de busca no panoramio deve ser o texto digitado pelo usuario.

  • Admin
A interface Admin apresenta o Backend da aplicação, permitindo o gerenciamento e edição dos dados de:
  1. usuários,
    • Admin::usuarios()
    • Admin::usuarios($id)
  2. pontos turísticos,
    • Admin::turisticos()
    • Admin::turisticos($id)
  3. hotéis,
    • Admin::hoteis()
    • Admin::hoteis($id)
  4. restaurantes,
    • Admin::restaurantes()
    • Admin::restaurantes($id)
  5. patrocinadores e
    • Admin::patrocinadores()
    • Admin::patrocinadores($id)
  6. parceiros
    • Admin::parceiros()
    • Admin::parceiros($id)

  • Taonde
A interface Taonde realiza as buscas.

Taonde comporta 2 metodos:

  1. restaurantes, que busca restaurantes

Taonde::restaurantes($busca)

  1. taqui, que busca hotéis e pontos turísticos

Taonde::taqui($busca)

Estes métodos devem ser chamados em ajax e carregados nas camadas específicas.
