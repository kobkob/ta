Pode carregar o controle taonde

Carregamos o taonde com new Taonde();

A melhor forma de garantirmos a presença do objeto do tipo Controller será testar a presença do router testando a existencia de

$taonde->router->method;

Taonde comporta 2 metodos:

  1. restaurantes, que busca restaurantes
  2. taqui, que busca hotéis e pontos turísticos

Pode usar o metodo restaurantes

Taonde::restaurantes()

A melhor forma de provar a chamada do metodo é testar o retorno com uma chamada de resposta conhecida Chamado-se o metodo com a chave 'Brasil' devera retornar 'comer Brasil'

Pode usar o metodo taqui

A melhor forma de provar a chamada do metodo é testar o retorno com uma chamada de resposta conhecida Chamado-se o metodo com a chave 'Brasil' devera retornar 'buscar Brasil'

Pode carregar o controle admin

Carregamos o admin com new Admin();

A melhor forma de garantirmos a presença do objeto do tipo Controller será testar a presença do router testando a existencia de

$admin->router->method;

Admin comporta 6 métodos

  1. usuários,
  2. pontos turísticos,
  3. hotéis,
  4. restaurantes,
  5. patrocinadores e
  6. parceiros
